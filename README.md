## Installation

1. Clone the repository:
   ```bash
   git clone <repository_url>
   cd thenece
   ```

2. Install dependencies:
   ```bash
   npm install or yarn install
   ```

3. Start the development server:
   ```bash
   npm run dev or yarn run dev
   ```
